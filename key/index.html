Public key & server address

If you're configuring RustDesk manually (iOS, Android, or if the pre-configured installer didn't work), paste these three values into the app's Settings → Network.

The address where RustDesk looks me up.

Relay server

Same address — handles fallback when direct connection isn't possible.

Public key

Identity proof — confirms you're connecting to my real server, not an imposter.

Where do I paste these?

  1. Open RustDesk.
  2. Tap or click the ⚙ Settings / three-dot menu.
  3. Go to Network.
  4. Paste each value into the matching field. Tap Apply.
  5. You're ready. Go back to the main screen, send me your ID + password.
